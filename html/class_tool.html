void Tool::handleEvents ( )

Handles user input.

LeftArrowKey - move character left
RightArrowKey - move character right
UpArrowKey - move character Up
DownArrowKey - move character down
Q - quit out of the game
Click-Down - start to edit a specific asteroid's location
Click-Up - finish editing a specific asteroid's location
B - return to main menu from any screen
D - enter demo mode
E - enter editor mode
